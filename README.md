# CPOS Home

CPOS Home is a free and open source home app (launcher) for Android. It is built on Lawnchair, which itself extends Launcher3, the default home app of the Android Open Source Project. CPOS Home keeps the customization and Pixel style experience of its upstream projects and ships pre installed on supported devices.

## About

CPOS Home brings a clean and familiar home screen together with rich customization. It adapts to your wallpaper and system theme, offers an at a glance area on the home screen, supports global search across apps and the web, and lets you adjust icons, fonts, colors, and grid layout to your liking. Because it is based on Launcher3 and Lawnchair, it stays close to stock Android behavior while adding the extra controls that power users expect.

## Building

CPOS Home uses the standard Gradle build of the upstream project. Clone the repository, open it in Android Studio or build from the command line with the included Gradle wrapper, and assemble the desired variant. Use gradlew on Linux and macOS, or gradlew.bat on Windows.
